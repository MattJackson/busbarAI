# Sudoku Web App

## app.py
A simple HTTP server using Python's `http.server` module that serves an interactive 9x9 Sudoku grid.

## solver.py
A backtracking solver for Sudoku that prints a solved board.

## How to Run
1. Run `python app.py` to start the server.
2. Open `http://localhost:8000/sudoku` in your browser to view the Sudoku grid.
3. Run `python solver.py` to solve a Sudoku board and print the solution.
